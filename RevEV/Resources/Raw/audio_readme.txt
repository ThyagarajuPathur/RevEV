RevEV Audio Sample Requirements
================================

Place your engine sound .wav files in this directory with the following names:

1. v8_muscle_loop.wav
   - V8 Muscle engine sample
   - Base recording RPM: ~3000 RPM
   - Should be seamlessly loopable
   - Format: 44.1kHz, 16-bit, mono or stereo

2. sport_loop.wav
   - High-revving sport/inline-6 engine
   - Base recording RPM: ~4000 RPM
   - Should be seamlessly loopable
   - Format: 44.1kHz, 16-bit, mono or stereo

3. futuristic_whine.wav
   - Electric/futuristic turbine sound
   - Base recording RPM: ~5000 RPM
   - Should be seamlessly loopable
   - Format: 44.1kHz, 16-bit, mono or stereo

Audio Sample Tips:
-----------------
- Record or obtain samples at a consistent RPM
- Ensure loop points are seamless (no clicks or pops)
- Normalize audio to -3dB for consistent volume
- Remove any background noise
- Mono is preferred for lower memory usage

Sources for royalty-free engine sounds:
- Freesound.org
- ZapSplat.com
- Soundsnap.com (subscription)
- Record your own from real vehicles

The app uses pitch-shifting to simulate different RPMs,
so a clean, mid-range sample works best as a base.
