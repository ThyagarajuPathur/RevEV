RevEV Font Requirements
======================

This app uses the Orbitron font family for its cyberpunk aesthetic.

Required font files:
-------------------
1. Orbitron-Regular.ttf
2. Orbitron-Bold.ttf
3. Orbitron-Black.ttf

Download Orbitron from:
- Google Fonts: https://fonts.google.com/specimen/Orbitron
- The font is free and open source (SIL Open Font License)

Installation:
------------
1. Download the Orbitron font family from Google Fonts
2. Extract the .ttf files
3. Place them in this directory (Resources/Fonts/)
4. Rename if necessary to match the names above

The app also uses OpenSans as a fallback font, which is included
in the default .NET MAUI template.

Note: Without the Orbitron fonts, the app will fall back to
the system default font, which may affect the visual appearance.
